Система Рекомендаций StarBank

    Spring Boot-сервис для выдачи персональных продуктовых рекомендаций клиентам.
    Основные возможности
    - Генерация рекомендаций на основе статических и динамических правил.
    - Управление правилами и сбор статистики через REST API.
    - Взаимодействие с пользователями через Telegram-бота.
